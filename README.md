# Crypto Volatility and Risk Analyzer

Welcome to **Crypto Volatility and Risk Analyzer**, a student development project undertaken as part of the **Infosys Springboard Internship Program**.

This project aims to analyze cryptocurrency price fluctuations and assess market risk using data analytics and statistical techniques. The objective is to convert volatile crypto market data into clear and meaningful insights that support informed decision-making.
